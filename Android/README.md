# Life Tracker Android App

Aplicación Android nativa desarrollada con Kotlin y Jetpack Compose (Material 3).

## Backend
Esta aplicación utiliza **Supabase** como backend para autenticación y base de datos.

## Configuración
Asegúrate de configurar tus credenciales de Supabase en el archivo de configuración correspondiente (se detallará en la implementación).

## Estructura
- **Autenticación:** Login y Registro via Supabase Auth.
- **Navegación:** Jetpack Navigation.
- **UI:** Material Design 3.
